# PHP Cafeteria Project

Simple overview of the main project structure:

```text
PHP-Cafetera/
├── app/
│   ├── config/            # Database and app configuration
│   ├── controllers/       # Request handlers/controllers
│   ├── models/            # Data models
│   └── views/             # UI pages (admin, auth, user, layouts)
├── public/
│   ├── index.php          # Front controller / entry point
│   ├── assets/            # CSS, JS, images
│   └── uploads/           # Uploaded files
├── routes/
│   └── web.php            # Application routes
├── cafeteria (1).sql      # SQL dump
├── cafeteria_db.sql       # Database schema
├── cafeteria_seed.sql     # Seed data
└── README.md
```
